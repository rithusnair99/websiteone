The Document Tree 
The elements in a web page are contained in a tree structure in which html is the root element that splits into the head and body elements 
The element is created by writing an opening tag, and completed by writing a closing tag
the opening and closing tags must be symmetrical
If an opening paragraph tag is followed by the opening element, the closing tags must appear in the reverse order
